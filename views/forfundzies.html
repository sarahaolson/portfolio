A crowdsource funding site where designers can fund their creative projects through direct donations and pledges from supporters.

MOTIVATION

This is a project that was made for my CS 3733: Software Engineering course. The aim was to make an application similar to kickstarter where designers can receive funding for their creative projects from supporters.

We learned how to use React and AWS in the first 3 weeks of the course and created a fully functional ForFundzies in the following 3 weeks!

FEATURES

All users have the ability to login to an existing account or create a new, non-admin account.

DESIGNERS
  • Create new projects
    • Add project name, story/description, genre, monetary goal, and deadline
    • Create/Modify/Delete pledges
  • Launch/Modify/Delete drafted projects
  • View progress of active projects
SUPPORTERS
  • Search for active projects
    • By genre
    • By typing in search bar
  • View active project pages
  • Support active projects
    • Direct donation amount
    • Though claiming pledges
  • View active pledges
ADMIN
  • View all projects (both active and inactive)
  • Delete projects of any status
